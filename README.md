## Installation

Add this line to your application's Gemfile:

    gem 'padma-assets'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install padma-assets

## Usage

In your application.css add:

    *= require padma-assets

In your application.js add:

    //= require padma-assets
